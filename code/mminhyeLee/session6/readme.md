## 새로 알게 된 점
히트맵을 통해서 서로 상관관계가 높은 독립변수들을 살펴보고, 다중공선성이 존재할 경우, 중첩되는 분포를 갖는 변수들을 제거해 회귀분석을 진행해야 한다는 것을 알게 되었습니다. 회귀분석의 전제(정규성, 독립성, 등분산성)들을 그래프를 그려 검정해보면서 회귀분석이 실효적인지 확인하는 방법을 알게 되었습니다.
